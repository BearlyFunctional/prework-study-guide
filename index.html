HTML

  • The head element contains information about the webpage, not visible to the user, including it's title. It should be placed before anything else, except doctype, which is always first.
  • Doctype, present at the beginning of the code, tells the web browser what to expect from the file.
  • The body element contains content for the user.
  • h1 through h6 are the tags for headers. The smaller the number the larger the text.
  • ul is the tag for bullet point lists, and should include the tag li to denote items in the list.

CSS

  • There are three methods of adding CSS: inline, integrated, and external.
  • Inline CSS: CSS code is added directly into the elements being edited.
  • Integrated CSS: Stylesheet is included in the HTML file's header.
  • External CSS: Seperate CSS file is created, and linked to with the "link" element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • cd: change directory.
  • cd ..: go up one directory.
  • pwd: print working directory, your current location in the file system.
  • ls: print list of files in current directory.
  • mkdir: Make directory, create new folder.
  • touch file-name: creates a file in your current directory.
  • start file-name: opens a file or application.
  • git status: checks what branch we are currently on.
  • git add -A: commit all file changes.
  • git commit -m "message here": commits current directory to chosen branch.
  • git checkout branch-name: changes current working branch.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin branch-name: pulls the latest file from the chosen branch.
  • git push origin branch-name: pushes the current local file to the chosen branch.

JavaScript

  • JavaScript and many other languages read from top to bottom, so order of commands is important.
  • An array is used to store multiple variables. When calling a variable from an array, the variables are numbered sequentially, starting with 0.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
